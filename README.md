# architecture-diagrams

System architecture diagrams for j13 Alaya stack.

All files are draw.io XML format. Open with:
- [draw.io online](https://app.diagrams.net) → File → Open from Device
- VS Code: install **Draw.io Integration** extension

## Files

| File | Description | Date |
|------|-------------|------|
| arch-alaya-full.xml | Full Alaya stack — all services, data flows, node details | 2026-03-21 |

## Convention

Filename format: `arch-{project-name}-{YYYYMMDD}.xml`
